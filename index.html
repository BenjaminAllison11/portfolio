Steam Genre Insight

The Steam Genre Distribution Site is a web application that analyzes a user's Steam library to provide insights into their game genre distribution, spending habits, and playtime allocation. The site also calculates genre diversity using Simpson's Diversity Index to help users understand their purchasing patterns.

Key Features

Retrieves a user's Steam library information via Steam's API

Calculates and visualizes genre distribution as a chart

Calculates Simpson's diversity index for genre variety assessment

Graph Coloring Algorithm Implementation

This project implements a graph coloring algorithm in Python, designed to solve the classic graph coloring problem. The algorithm determines the minimum number of colors required to color the vertices of a graph such that no two adjacent vertices share the same color. This implementation utilizes a backtracking approach to explore possible color assignments and efficiently find a valid solution.

Key Features

The graph is represented using an adjacency matrix, allowing for flexible representation of various graph structures.

Implements a recursive backtracking algorithm to explore different color combinations.

Includes a "safe" function to check if a color assignment is valid.
